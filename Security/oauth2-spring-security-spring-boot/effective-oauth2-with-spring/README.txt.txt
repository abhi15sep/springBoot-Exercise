GIT location of the source: https://github.com/wlesniak/effective-oauth2-with-spring-security-and-spring-boot.git

This demo uses Lombok, for annotations like @Getter and @Setter to speed up development, you can install it @
https://projectlombok.org


Any issues please feel free to message me in the in the dicussion.
From modules 4 - 8 , you will need an instance of a keycloak server running on port: 8081
You will also need to create all the users, clients, roles etc, as outlined in the modules.
I have included the: "realm-export.json" file, which you can import into your keycloak instance, 
this will setup everything for you.
In the admin console of keycloak: under Manage: Import.
